NOMBRE DEL PROYECTO:
    WHM-Mailguard - Plugin de Protección de Correo para WHM

DESCRIPCIÓN:
    Este proyecto es un plugin para Web Host Manager (WHM) que proporciona
    funcionalidades de protección y gestión de correos electrónicos.

================================================================================
ESTRUCTURA DEL PROYECTO:
================================================================================

├── mailguard.tmpl                # Plantilla de configuración
├── install.sh                    # Script de instalación del plugin
├── uninstall.sh                  # Script de desinstalación del plugin
├── README.md                     # Documentación principal (vacío)

├── backend/                      # Componente backend - Lógica principal

================================================================================
COMPONENTES PRINCIPALES:
================================================================================

1. FRONTEND (whm-plugin/)
   - mailguard.cgi: Interfaz web accesible desde el panel de control WHM
   - mailguard.conf: Configuración específica del plugin

2. BACKEND (backend/)
   - mailguard.py: Servicio principal que ejecuta la lógica de protección
   - mailguard.service: Permite que el servicio se inicie automáticamente
   - schema.sql: Define la estructura de almacenamiento de datos

3. INSTALACIÓN (scripts)
   - install.sh: Automatiza la instalación del plugin en WHM
   - uninstall.sh: Automatiza la desinstalación del plugin


================================================================================
TECNOLOGÍAS UTILIZADAS:
================================================================================

Backend:
    - Python (mailguard.py)
    - SQL (para base de datos)
    - Systemd (para gestión de servicios en Linux)

Frontend:
    - CGI Script (para interfaz web en WHM)
    - Bash (install/uninstall scripts)

Sistema Operativo Objetivo:
    - Linux (basado en la presencia de .service y .sh)
